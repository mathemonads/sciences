\section{Mathematics}

A \emph{curve} in a topological space $X$ is defined to be a continuous function from an interval in $\R$ to $X$.
Recall that a curve in a topological module is \emph{nonvanishing} if it is nowhere equal to the identity $\vzero$ of addition.

\subsection{Totally ordered field of real numbers}

Denote the totally ordered field of real numbers as follows.
\begin{align*}
	\left\langle \R, \R \times \R \xrightarrow{+} \R, 0, \R \times \R \xrightarrow{\cdot} \R, 1, \leq \right    \rangle
\end{align*}
